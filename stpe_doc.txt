## GLOBAL STRUCTURE

• **Layouts:**
  - `src/components/layout/Navbar.tsx`: Fixed top navigation with landing/app variants.
  - `src/components/layout/BottomNav.tsx`: Bottom navigation for mobile apps.
  - `src/components/layout/Sidebar.tsx`: Sidebar navigation for dashboard.

• **Navigation systems:**
  - `React Router` used in `App.tsx` handling protected routes and layout switching.

• **Global components:**
  - `ScrollToTop.tsx`: Resets scroll on route change.
  - `ProtectedRoute.tsx`: Route guard checking auth status.
  - `NotificationHandler.tsx`: Listens to socket notifications.

• **Providers / Contexts:**
  - `AuthContext.tsx`: Authentication state.
  - `FriendsContext.tsx`: Friends list and status.
  - `GamesContext.tsx`: Games data.
  - `LobbyContext.tsx`: Lobby state management.
  - `ProfilePopoverContext.tsx`: Manages popover visibility.

• **Global UI systems:**
  - `LobbyOverlay.tsx`: Global overlay for lobby state.
  - `FriendChatOverlay.tsx`: Global chat interface.
  - `Toast.tsx`: Notification system (via `react-hot-toast`).
  - `CreateLobbyModal.tsx`: Global lobby creation modal.
