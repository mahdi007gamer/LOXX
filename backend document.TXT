LOXX BACKEND TECHNICAL SPECIFICATION v2.0

Enterprise‑Grade Architecture

Real‑Time Gaming Social Platform for Persian-speaking Users

OVERVIEW
LOXX is a real‑time gaming social platform designed primarily for Persian-speaking users.

All UI and user-generated content is RTL (right-to-left) and Persian.

Backend services, logs, internal architecture, and APIs are English for international developer compatibility.

All features must support Persian usernames, Persian text in chat, and Middle-East/Iran regional routing.

Platform Goals:

Real-time text chat and voice chat with ultra-low latency
Persian-focused experience (Farsi text, right-to-left UI, Iran/Middle East regions)
Scalable social + lobby platform similar to Discord + Steam
Strong security (JWT rotation, Argon2id hashing, anti-abuse systems)
Analytics, monitoring, and audit logs suitable for production
Voice chat powered by Mediasoup SFU
Realtime system powered by Redis and Socket.io
Highly stable and scalable for tens of thousands of concurrent users
AUTHENTICATION SYSTEM
Primary Requirements:

Fully secure authentication
Persian names and inputs allowed

Security Requirements:

Password hashing: Argon2id
Access Token: 15 minutes
Refresh Token: 7 days
Refresh Token Rotation: previous token invalidated in Redis
Refresh Token Blacklist: redis key refresh_blacklist:{tokenId}
Login Rate Limit: 5 attempts / 15 minutes / IP
Account Lockout: 10 failed attempts → 30 minute lock
Sensitive Actions Logged:

Login (success + failure)
Password change
Email change
Token refresh abuse
Suspicious IP change
Logout
All stored in audit_logs table.
